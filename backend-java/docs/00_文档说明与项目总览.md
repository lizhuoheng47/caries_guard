# 00 文档说明与项目总览

## 1. 文档定位

本目录只保留 Java 后端长期设计和代码入口说明。阶段过程文档、E2E 临时清单、Postman 大量过程用例已移除。

## 2. Java 后端定位

Java 是 CariesGuard 的业务主链：

- 用户、组织、权限；
- 患者、就诊、病例；
- 影像和附件；
- analysis task；
- callback 落库；
- 报告、复核、随访；
- RAG 展示入口。

Python 提供 AI/RAG 能力。

## 3. AI 路线

当前采用通用大模型 + 知识图库 RAG，不采用业务专用大模型微调作为主路线。

Java 不直接调用大模型 provider，不保存 prompt 全量敏感内容，不用 RAG 结果直接改变临床业务状态。
