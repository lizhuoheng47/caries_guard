# 文档说明与项目总览

更新日期：2026-04-15

本目录只保留当前项目最终参考文档，不保留开发过程记录。内容以 `backend-java` 当前代码、Flyway V014 和已实现功能为准。

## 1. 当前结论

1. Java 后端已实现患者、就诊、病例、影像、AI 分析、医生修正、报告、随访、看板、系统权限的业务闭环。
2. MinIO 已保留并开发完成，`application-local.yml` 默认 `provider-code=MINIO`，实现类为 `MinioObjectStorageService`。
3. `LOCAL_FS` 仍保留为 e2e 和本地兼容 provider，实现类为 `LocalObjectStorageService`。
4. AI 请求载荷 `AiAnalysisRequestDTO` 已包含 `accessUrl`、`accessExpireAt`、`storageProviderCode`、`attachmentMd5`、`localStoragePath`。
5. AI 回调契约为 `AiAnalysisResultCallbackCommand`。
6. V014 已新增 `ana_model_version_registry`、任务 trace/耗时字段、修正反馈训练准入字段、业务角色、菜单、角色菜单关系和数据权限规则。
7. 报告导出接口返回审计结果和下载信息：`attachmentId`、`downloadUrl`、`expireAt`。
8. `Risk` 是风险评估能力，不是独立模块；`ModelAdmin` 保留治理原则，但当前未落地为独立业务模块。

## 4. 外部依赖
