## 3.2 API 契约差异

`03_核心API契约与模块接口规范.md` 描述的是目标接口，不是当前已实现接口。

当前已实现接口与目标契约存在差异：

### 登录接口
当前实现返回：
- `tokenType`
- `accessToken`
- `expiresIn`

目标文档返回：
- `token`
- `expireIn`
- `user`

### `/me`
当前实现返回：
- `userId`
- `orgId`
- `username`
- `displayName`
- `userTypeCode`
- `roleCodes`

目标文档返回还包含：
- `permissions`
- `nickName`

### `/ping`
当前实现返回：
- `{"status":"ok"}`

目标文档期望：
- `{"pong": true, "app": "caries-guard-backend"}`

因此：

> `03` 文档适合作为“后续接口冻结目标”，不应被解读为“当前接口已经符合”。
