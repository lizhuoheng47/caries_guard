## 6. 当前未覆盖边界声明

- 当前 `MainlineWorkflowE2ETest` 使用 `MockMvc(addFilters = false)`，验证的是控制器到数据库的真实业务链，不是完整 JWT 过滤器链
- 当前对象存储仍使用 bean 级 mock，不接真实 MinIO
- 当前 MQ 仍保持 logging publisher，不接真实 RabbitMQ 消费链
- 当前尚未形成“部署脚本 + 演示环境一键启动”交付件

这些属于 P8 后续收口项，不影响当前主链路验证成立。
