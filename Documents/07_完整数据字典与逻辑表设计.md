# 07 完整数据字典与逻辑表设计

## 1. 数据库分组

| 数据库 | 说明 |
| --- | --- |
| `caries_biz` | Java 业务数据 |
| `caries_ai` | Python AI/RAG 运行数据 |

| 表 | 说明 |

| 表 | 说明 |

## 4. AI Summary 字段

`ana_result_summary` 应稳定保存：

- `overall_highest_severity`
- `uncertainty_score`
- `review_suggested_flag`
- `summary_json`
- `raw_result_json`

`raw_result_json` 保存扩展字段。

## 5. Phase 5C 字段

必须在 JSON 中保留：

- `gradingMode`
- `gradingImplType`
- `gradingLabel`
- `confidenceScore`
- `uncertaintyMode`
- `uncertaintyImplType`
- `uncertaintyScore`
- `needsReview`

## 6. RAG 字段

RAG 相关字段：

- `knowledge_base_code`
- `document_code`
- `chunk_text`
- `embedding_model`
- `retrieval_score`
- `llm_provider_code`
- `llm_model_name`
- `citations_json`
- `safety_flags_json`

## 7. 附件字段

附件必须包含：

- `bucket_name`
- `object_key`
- `content_type`
- `file_size_bytes`
- `md5`
- `storage_provider_code`
- `visibility_code`

MinIO object 不替代数据库附件主数据。
